Proyectos Destacados

Tienda Virtual (E-commerce B2C Escala Nacional)

La Tienda Virtual es una plataforma de comercio electrónico a escala nacional construida con React (y probablemente Redux para el estado), donde desarrollé todas las funcionalidades frontend clave, incluyendo un carrito de compras eficiente, un proceso de checkout optimizado y la simulación de integración con pasarelas de pago para una experiencia transaccional fluida y segura. El enfoque principal fue la experiencia de usuario (UX) y el diseño responsivo utilizando Bootstrap 5, así como la implementación de técnicas de Optimización SEO para maximizar el alcance de los productos..

Tech: React, Redux, Bootstrap 5

Tarjeta de Proyecto Concesionario Online

Este proyecto se centra en el desarrollo de un Dashboard de Analíticas Empresariales que simula un sistema de gestión para un concesionario online, permitiendo la visualización de datos de inventario y ventas en tiempo real mediante gráficos interactivos y tablas dinámicas. Demuestra mi capacidad para manejar y estructurar interfaces ricas en datos, aplicando HTML5, CSS3 Grid y Vanilla JavaScript con librerías de gráficos, asegurando además que la aplicación sea robusta y cumpla con altos estándares de accesibilidad (ARIA) para todos los usuarios..

Tech: HTML5, CSS3 Grid, Vanilla JS (Charts.js)

App Móvil Progresiva (PWA)

Este proyecto fue desarrollado como una Aplicación Móvil Progresiva (PWA), ideal para una Clínica Dental Online, que permite a los pacientes gestionar citas y acceder a información con la fiabilidad de una aplicación nativa. La PWA fue implementada con React (o React Native Web) utilizando Service Workers para habilitar la funcionalidad offline, el envío de notificaciones push para recordatorios de citas y un diseño Mobile-First, demostrando mi habilidad para crear soluciones rápidas, accesibles y disponibles en múltiples dispositivos..

Tech: React, Service Workers, PWA
